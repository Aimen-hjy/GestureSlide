# 基于MediaPipe的无接触手势PPT控制系统

通过摄像头实时识别手势，实现无接触控制PPT演示。支持翻页、播放、暂停、鼠标移动点击、音量调节等功能。

## 功能一览

### 普通模式
| 手势 | 动作 | 说明 |
|------|------|------|
| 👈👈 挥手(左/右) | ← / → | 上一页/下一页 |
| ✊ 握拳 | B键 | 切换黑屏/恢复 |
| 👌 OK手势 | F5 | 开始PPT放映 |
| ☝️ 食指指向 | 切换模式 | 进入鼠标控制模式 |
| ✌️ V手势 | 音量+ | 调高系统音量 |
| 🤟 三指伸展 | 音量- | 调低系统音量 |
| 👍 竖拇指 | F5 | 开始PPT放映 |
| 🤙 仅小指 | Esc | 退出PPT放映 |

### 鼠标模式
| 手势 | 动作 | 说明 |
|------|------|------|
| ☝️ 移动食指 | 移动光标 | 食指控制鼠标指针 |
| 🤏 拇指食指捏合 | 左键点击 | 模拟鼠标点击 |
| 🖐 五指张开 | 退出 | 返回普通模式 |
| ✊ 握拳 | 退出 | 返回普通模式 |

## 环境要求

- Python 3.8+
- Windows / macOS / Linux
- 摄像头 (内置或USB)

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 使用步骤

1. 确保摄像头正常工作
2. 打开你的PPT文件
3. 运行本程序
4. 使用手势控制PPT：
   - 做 **OK手势** 开始放映
   - **左右挥手** 翻页
   - **食指指向** 进入鼠标模式，移动食指控制光标
   - **握拳** 在放映时切换黑屏
   - 按 **q键** 退出程序

### 4. 配置调整

编辑 `config.py` 可调整：
- 摄像头分辨率和帧率
- 手势识别灵敏度
- 滑动触发阈值
- 鼠标移动速度
- UI显示样式

## 技术原理

### 手部检测
使用 Google MediaPipe Hands 模型，实时检测21个手部关键点坐标。

### 手势识别
通过分析手指关节几何关系判断每根手指的伸展/弯曲状态，组合识别11种预定义手势。

### 滑动检测
维护手腕位置的历史记录滑动窗口，累计位移超过阈值时触发翻页。

### 防抖机制
- 手势需持续N帧才确认触发（避免瞬时误识别）
- 每种手势有独立冷却时间（避免重复触发）
- 鼠标移动采用指数平滑（避免抖动）

### 鼠标映射
食指指尖的相对位移按速度倍率映射为屏幕鼠标移动，带死区过滤和平滑处理。

## 依赖库

- **opencv-python** - 摄像头采集与图像处理
- **mediapipe** - Google手部关键点检测
- **pyautogui** - 键盘鼠标事件模拟
- **numpy** - 数值计算

## 注意事项

1. 保持手部在摄像头视野内，光照充足时效果最佳
2. 手势动作要清晰、稳定，避免过快晃动
3. 鼠标模式下移动食指即可控制光标，不需要大幅度移动
4. 程序使用 `pyautogui.FAILSAFE`，将鼠标移到屏幕角落可紧急停止
5. 如遇摄像头无法打开，请检查 `config.py` 中的 `CAMERA_ID` 设置
